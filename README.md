Expense Tracker is a RESTful backend application developed using Spring Boot that helps users manage their daily expenses.  
It allows users to add, view, and delete expense records efficiently.

---

## Tech Stack
- Java  
- Spring Boot  
- Spring MVC  
- Spring Data JPA  
- MySQL  
- Maven  

---

## Features
- Add new expense  
- View all expenses  
- Delete expense by ID  

---

## API Endpoints
| Method | Endpoint | Description |
|------|---------|-------------|
| POST | /expenses | Add a new expense |
| GET | /expenses | Get all expenses |
| DELETE | /expenses/{id} | Delete an expense |

---

## Testing
All APIs are tested using **Postman** to ensure proper functionality.
